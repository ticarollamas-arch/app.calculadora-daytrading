# app.calculadora-daytrading
Ferramenta interativa desenvolvida em JS para controle de juros compostos e gestão de banca. Possui "Balança Inteligente" que recalcula metas dinamicamente após perdas. Focada em estratégias sem Martingale, priorizando a consistência e a sobrevivência do capital a longo prazo. 100% "Pé no Chão".
